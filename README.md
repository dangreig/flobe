# Flobe

Flobe is a full-screen dynamic background playground for focus, ambience, and visual experimentation.

It runs as a single-page canvas app with a collection of animated scenes, adjustable palettes, speed and density controls, fullscreen mode, UI hiding, FPS display, and a built-in focus timer.
